# Prompt 02: Inserción de Datos Semilla (DML)

Actúa como un desarrollador backend para el proyecto "FoodCampus". 
Genera un script SQL que:
1. Inserte al menos 15 restaurantes con temática universitaria (ej: "Comedor Central UTM", "Pizza de Ingeniería").
2. Inserte al menos 5 pedidos de prueba en la tabla `Pedido` incluyendo un nombre de usuario real (ej: 'JuanPerez', 'MariaGomez').
3. Inserte los detalles de esos pedidos vinculando `PedidoId` y `RestauranteId` existentes.
4. Asegurar que los datos sean variados en fechas y costos de envío.

Genera solo el código SQL encapsulado en un bloque de código.
